# Practice
最后一个单词的长度
## 问题分析：
#### 
给定一个字符串， 包含大小写字母、空格 ' '，请返回其最后一个单词的长度。

如果不存在最后一个单词，请返回 0 。

注意事项：一个单词的界定是，由字母组成，但不包含任何的空格。
## 编程实现：
```C++
class Solution {
public:
    int lengthOfLastWord(string s) {
      int a=0,b=s.length()-1;
        while (b>=0&&s[b]==' ') b--;
        while (b>=0&&s[b]!=' ')
        {
            a++;
            b--;
        }return a;
         
    }
    };
```
