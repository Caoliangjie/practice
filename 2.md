# practice
 2的幂
## 问题分析：
#### 
给定一个整数，写一个函数来判断它是否是2的幂
## 编程实现：
'''C++
class Solution {

public:
    
    bool isPowerOfTwo(int n) {
       
       if (n==0)
          
          return false;
        
        while (n % 2 == 0) 
         
         n /= 2;
       
       return (n == 1);
   
   }

};
```
## 总结体会：
这个算是比较简单的做法，我的想法就是无限除二，最后看是不是能整除，然后输出true或者false保证是布尔数即可。今天学了位与运算后，发现也有一个用n&n-1的位与做法，这个与2的幂的二进制码有关，也是一个很简单的做法。
